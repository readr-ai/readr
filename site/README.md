# Readr launch site

Single-page landing site for Readr (self-contained `index.html` + optimized screenshots in `assets/`).
Deployed to GitHub Pages by `.github/workflows/pages.yml` on every push to `main` that touches `site/**` (or manually via workflow_dispatch).
Expected URL: https://readr-ai.github.io/readr/
If the first deploy fails, enable Pages once in repo Settings → Pages → Source: "GitHub Actions".
